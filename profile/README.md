# Untitled Sekai

こんにちは。このOrganizationはUntitled Sekaiという「プロセカ」の創作譜面投稿プラットフォームを作成するために設立されました。

## 目的
- 投稿プラットフォームを作成
- 創作譜面に便利なツールを作成

## 開発方針
- ブランチは`main`と`dev`を分けることが多数です
- Issueをベースに開発します
